Proyecto destacado

Helpdesk

Creación de sistema de reporte de incidencias.

Descripción del Proyecto

Este proyecto consistió en la creación de un Helpdesk que pone a disposición del usuario un formulario para reportar distintos tipos de incidencias técnicas.

En este caso, llevé a cabo la creación de un formulario en HTML preparado para conectarse a una base de datos en MySQL, la cual a su vez alimenta un documento de Access donde se guardan los registros de forma local.

Tecnologías y Herramientas

VSCode
MySQL
Microsoft Access
