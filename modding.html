Modding de Ordenador

Construcción de una caja de ordenador personalizada.

Descripción del Proyecto

Este proyecto grupal consistió en la construcción de una caja para ordenador con materiales recicables y de bajo coste, usando temática libre.

En este caso, adaptamos los componentes de un ordenador a una caja con forma de caravana, cuyas partes fueron cortadas a medida. Se añadieron detalles como extractores para el flujo de aire, una llave de encendido e iluminación interna.
