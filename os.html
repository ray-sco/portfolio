Proyecto destacado

Gestión de SO

Instalación y configuración de servicios esenciales en sistemas operativos locales y en red.

Descripción del Proyecto

Esta sección recoge puntos clave del proceso de instalación y configuración de servicios en sistemas operativos Windows y Linux, tanto para usuarios individuales como servidores en red.

En este caso, las actividades realizadas incluyen la instalación singular o simultánea (Dual Boot) de dichos sistemas, así como la gestión de roles y características como Active Directory, DHCP, DNS, Proxy, Radius, sistemas de redundancia y copias de seguridad.

Tecnologías y Herramientas

Windows 10/11/Server
Linux Ubuntu Server
Raspberry Pi
